# XpressBuy247 Suite — GitHub Workshop (vX.0.0.2)

**PHP:** 7.4–8.3 · **WP:** 6.8+ · **Architecture:** Core → Guardian → Sandbox → Watchdog

This repository auto-builds verified, install-ready WordPress plugins.

## Quick Start
1) Create an empty GitHub repo named `xpressbuy247-suite`.
2) Upload the contents of this folder (the 6 items) to the repo root.
3) Open the repo → **Actions** tab. The workflow runs automatically.
4) Download **Artifacts** when the build finishes (green check).
5) Install ZIPs in WordPress: Core → Guardian → Sandbox → Watchdog.
