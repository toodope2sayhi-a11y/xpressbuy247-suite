# Visual Guide — Upload These 6 Items (vX.0.0.2)

Upload to the ROOT of your GitHub repo:

- `.gitignore`
- `README.md`
- `Visual_Guide.md`
- `.github/` (contains workflows/build-and-package.yml)
- `builder/` (contains build.php)
- `plugins-src/` (contains the 4 plugin folders)

Then go to **Actions** → wait for ✅ → download **Artifacts** (your ZIP plugins).
